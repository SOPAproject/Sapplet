Let's enjoy panoramic audio!

This repository provides source code of a Java applet 'Sapplet.class' and its related files. Sapplet.class is an applet for reproduction of a SOPA file.

SOPA that stands for 'Stream Of Panoramic Audio' is a custom audio data format proposed by us and can be used for transmitting panoramic audio signals online. This format allows the listener to control the panning during the reproduction of sounds.


Demonstration of a SOPA file.

To start the demonstration, click the line 'CLICK to start' in the applet below. During the reproduction, the panning of sound can be controlled by scrolling (dragging) imageview to the left or the right.

You need Java on your PC to run the applet.


Detail description about the SOPA project and some of its demonstration are provided in the following Web site.

Panoramic audio Web site

( http://staff.aist.go.jp/ashihara-k/pan_top.html )
